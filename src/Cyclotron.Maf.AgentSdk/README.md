# Cyclotron.Maf.AgentSdk

A .NET SDK for building AI agent workflows using Microsoft Agent Framework (MAF) and Azure AI Foundry.

## Features

- **Workflow Orchestration**: Build complex AI agent workflows with ease
- **Agent Factories**: Create and manage AI agents using Azure AI Foundry
- **Vector Store Management**: Seamlessly integrate with vector stores for document retrieval
- **PDF Processing**: Process and extract content from PDF documents
- **OpenTelemetry Integration**: Built-in observability with OpenTelemetry support
